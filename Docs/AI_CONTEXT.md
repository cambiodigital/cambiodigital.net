# Contexto IA del proyecto

## Propuestas y presupuestos

- Las propuestas comerciales y presupuestos por cliente se guardan en `presupuestos/[Cliente o tema]/`.
- Para nuevas propuestas, usar Markdown como formato base cuando el entregable sea documental.
- Mantener encabezado de Cambio Digital, cliente, fecha, validez, resumen ejecutivo, alcance, inversion, supuestos, limites y recomendacion final.
- Los precios deben derivarse de `Docs/Lista de precios.md` o de presupuestos comparables ya existentes en `presupuestos/`.
- Si el alcance depende de requisitos pendientes, separar diagnostico/validacion de implementacion para evitar cerrar presupuestos a ciegas.
